¿Por qué necesitamos Layer 2?

Ethereum es como una autopista de un solo carril muy segura pero lenta. En horas pico, el peaje (Gas) sube a $50 USD.

Las **Layer 2 (L2)** son autopistas laterales que procesan miles de transacciones por segundo y luego envían un "resumen" a la autopista principal.

Beneficios Clave:
  • • Gas Fees 100x más baratos.
  • • Confirmación instantánea (UX de Web2).
  • • Heredan la seguridad de Ethereum.

La Guerra de los Rollups

🐢

Optimistic Rollups

"Asumen que todos son honestos". Si alguien miente, hay un periodo de 7 días para impugnar la transacción. Es simple y compatible con todo el código de Ethereum.

Integración en el Frontend

Para que tu App use una L2, debes pedirle al usuario que cambie de red en MetaMask. Aquí está el código estándar para añadir Arbitrum: 

async function addArbitrum() {
  await window.ethereum.request({
    method: 'wallet_addEthereumChain',
    params: [{
      chainId: '0xa4b1', // 42161 en decimal
      chainName: 'Arbitrum One',
      nativeCurrency: { name: 'ETH', symbol: 'ETH', decimals: 18 },
      rpcUrls: ['https://arb1.arbitrum.io/rpc'],
      blockExplorerUrls: ['https://arbiscan.io/']
    }]
  });
}
