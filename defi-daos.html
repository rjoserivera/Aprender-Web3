DeFi: El Motor del Capital

La Magia del AMM

Los **Automated Market Makers (AMM)** como Uniswap no usan libros de órdenes. Usan una fórmula matemática constante para determinar el precio:

x * y = k

Donde x e y son las cantidades de dos tokens en el pool, y k es una constante fija.

Si alguien compra Token X, la cantidad de X disminuye, por lo tanto el precio de X debe subir para que k se mantenga igual. ¡Oferta y demanda pura en código!

Ejemplo de Lógica de Swap 
// Simplificación de un Swap
function swap(uint amountIn, address tokenIn) public {
    uint reserveIn = getReserve(tokenIn);
    uint reserveOut = getReserve(tokenOut);
    
    // x * y = k
    // (x + dx) * (y - dy) = k
    uint amountOut = (reserveOut * amountIn) / (reserveIn + amountIn);
    
    IERC20(tokenIn).transferFrom(msg.sender, address(this), amountIn);
    IERC20(tokenOut).transfer(msg.sender, amountOut);
}
Liquidity Provision

Tú puedes ser el banco. Al depositar tus tokens en el pool, ganas una comisión por cada swap que otros hagan.

Impermanent Loss

El riesgo de DeFi. Si el precio de un token cambia mucho respecto al otro, podrías haber ganado más simplemente guardándolos.

¿Por qué es revolucionario?

1. Transparencia Radical: En una DAO, puedes ver exactamente cuántos fondos hay en la caja y en qué se están gastando. No hay "gastos reservados".

2. Resistencia a la Censura: Un protocolo DeFi no puede discriminarte por tu país, crédito o religión. Si tienes los tokens, puedes usar el servicio.

3. Alineación de Incentivos: Si el proyecto crece, el valor de tu token de gobernanza sube. Todos trabajan para el bien común del protocolo.

4. Ejecución Atómica: En DeFi, el intercambio de tokens ocurre en un solo paso. O se hace todo, o no se hace nada. Cero riesgo de estafa.
