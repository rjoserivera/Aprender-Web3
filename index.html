Los 3 Pilares de Web3

Lo que el jurado de una hackathon espera que domines.

🏠

Propiedad (Ownership)

En la Web2, tus datos pertenecen a Facebook o Google. En Web3, tú eres el único dueño de tu información mediante Llaves Criptográficas.

🛡️

Descentralización

No hay un servidor central. La base de datos (Blockchain) vive en miles de computadoras. Si una falla, el sistema sigue vivo.

🤝

Confianza (Trustless)

No necesitas "confiar" en una empresa. Confías en el Código Inmutable (Smart Contracts) que se ejecuta exactamente como fue escrito.

Backend (Blockchain Logic)

Así es como la red responde a tu petición de balance:

// Solidity: Estado Global de Ethereum
mapping(address => uint256) private _balances;

// El nodo busca tu dirección 0x... en este mapeo
function balanceOf(address account) public view returns (uint256) {
    return _balances[account]; 
}

Backend (Consensus Validation)

Antes de mover 1 wei, los mineros verifican esto:

// Validación del Nodo
require(msg.sender.balance >= amount, "Saldo insuficiente");
require(verifySig(msg.hash, signature), "Firma inválida");

// Si es válido, se actualiza el estado atómico
balances[from] -= amount;
balances[to] += amount;
